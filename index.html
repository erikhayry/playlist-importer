To save a playlist

Select all the songs from a playlist and hit copy. Paste what you got in a text editor unformatted and save as text file. Should look like this with each song on a row:

To import a playlist

Copy the text from you backupfile and paste it into the Playlist Importer and click 'Import playlist'. If the format is right a playlist will be created
